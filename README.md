model层——提供实体类映射数据库记录、       
util层——dbcp线程池连接MySQL数据库、        
dao层——dbutils封装JDBC代码简化sql语句、       
service层、        
controller层servlet、         
前台页面jsp。        
STL JSP Standard Tag Library 标准标签库,JSTL允许开人员可以像使用HTML标签 那样在JSP中开发Java功能。
